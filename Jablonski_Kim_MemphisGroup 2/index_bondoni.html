Circus Icons

bodoni font

The objective of this project was to use elements directly taken from the letters of the Bodoni font. Each icon uses several weights and pieces that represent the characteristics of Bodoni. The icons I created represented a circus theme involving the big top tent, a clown and character riding a unicycle. Once icons were created I developed a visual language for them to be represented through. Using primary colors and playful lock ups, allowed for them to display the bright and playful circus feeling.
